# 🔐 Secure Authentication System

A modern Flask-based authentication system designed with security, scalability, and user experience in mind. This project demonstrates industry-standard authentication practices including JWT authentication, password hashing, role management, dashboard UI, and account security features.

## 🚀 Features

* User Registration
* User Login
* Password Hashing using bcrypt
* JWT Authentication
* Secure Session Management
* Responsive Dashboard
* Modern Sidebar Navigation
* Forgot Password Interface
* OTP Verification Interface
* User Role Management
* Login Activity Tracking Structure
* Professional Flask Project Architecture

## ⚙️ Installation

### Clone Repository

```bash
git clone <repository-url>
cd Secure-Authentication-System
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Run Project

```bash
python app.py
```

Open:

```text
http://127.0.0.1:5000
```
